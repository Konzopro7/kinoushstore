SEO Checklist (production)

Utilise cette page pour valider rapidement le referencement apres mise en ligne.

Indexation et sitemaps

  • Verifier https://kinoushstore.com/sitemap.xml
  • Verifier https://kinoushstore.com/robots.txt
  • Soumettre le sitemap dans Google Search Console
  • Lancer une demande d'indexation pour la page d'accueil

Meta et partage social

  • Verifier les balises title/description par page
  • Verifier OG/Twitter avec l'outil de debug Facebook/X
  • Valider l'image OG (1200x630)
  • Verifier les pages noindex (panier, paiement)

Performance

  • Tester PageSpeed (mobile + desktop)
  • Compresser les images produits (WebP si possible)
  • Activer la mise en cache CDN

Google Search Console
